# 📊 Pesquisa de Satisfação

Este projeto foi desenvolvido com o objetivo de realizar uma **pesquisa de opinião** com seus clientes e avaliar o grau de satisfação no atendimento.

---

## 🎯 Objetivo

- Coletar informações de **50 entrevistados**, incluindo:
  - Nome
  - Idade
  - Opinião sobre o atendimento:
    - `1` → EXCELENTE  
    - `2` → BOM  
    - `3` → RUIM  

- Ao final da pesquisa, o programa deve mostrar:
  - Quantidade de respostas **EXCELENTE**
  - Quantidade de respostas **RUIM**

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- Estruturas de repetição (**FOR**)
- Estruturas de decisão (**if/elif/else**)
